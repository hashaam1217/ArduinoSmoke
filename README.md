# ArduinoSmoke
This is a smoke alarm that detects the air quality and turns on ventilation. 
It has 4 stages: 
1. Clean Air    (Fan OFF, Alarm OFF) 
2. Polluted Air (Fan OFF, Alarm OFF)
3. Polluted Air (Fan ON,  Alarm ON)
4. HAZARD       (Fan OFF, Alarm ON)
